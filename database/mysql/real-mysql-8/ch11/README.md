# 11. 쿼리 작성 및 최적화

쿼가 빠르게 수행되게 하려면 데이터베이스 서버에서 쿼리가 어떻게 요청을 처리할지 예측할 수 있어야 한다. 그래서 SQL을 작성하는 방법이나 규칙은 물론 내부적인 처리 방식(옵티마이저)에 대해 어느 정도의 지식이 필요한 것이다.

이번 장에서는 쿼리의 패턴별로 "어떻게 처리되는가?" 를 살펴본다.

## 목차

1. 쿼리 작성과 연관된 시스템 변수
2. 매뉴얼의 SQL 문법 표기를 읽는 방법
3. MySQL 연산자와 내장 함수
4. SELECT
5. INSERT
6. UPDATE와 DELETE
7. 스키마 조작(DDL)
8. 쿼리 성능 테스트
